# Wacht Documentation

Wacht is distributed uptime monitoring for self-hosted deployments. It ships as
Docker images for the server, probes, and web UI. It runs HTTP, TCP, and DNS
checks from multiple probes, opens incidents from probe agreement, sends
webhook alerts, and exposes a public status page.

## Start Here

- Install Wacht with Docker Compose.
- Start the server first and add probes from the dashboard.
- Configure server, probe, and check settings.
- Add or run probes.
- Understand quorum alerts and public status pages.
- Operate the self-hosted stack.

## Documentation Types

The docs follow the Divio/Diataxis structure:

- **Tutorial:** first install
- **How-to guides:** probes and operations
- **Reference:** configuration
- **Explanation:** alerts, quorum, and status pages
